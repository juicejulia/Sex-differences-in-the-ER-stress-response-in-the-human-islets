# Sex differences in the ER stress response in the human islets

This repository hosts the codes to generate analysis and figures for Sex differences in the ER stress response in the human islets.

## Pipeline
1. Background removal: 01_SoupX.R
2. Integration and celltype calling: 02_Integration_celltype_calling.R
3. Differential expression: 03_Differential_expression_pseudobulk.R
4. AddModuleScore for UPR pathway markers: 04_UPR.R
5. Slingshot pseudotime analysis: 05_Pseudotime.R
6. XBP1_GoT
